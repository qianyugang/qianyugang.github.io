介绍

提供的CLI（命令行界面Command Line Interface）允许启动，停止和管理Kong实例。CLI可以管理本地节点（如在当前计算机上）。

如果您还没有使用，我们建议您阅读配置参考。

通用标志参数

所有命令都将一组特殊的可选标志作为参数：

  • --help：打印此命令的帮助信息
  • --v：启用详细模式
  • --vv：启用调试模式（很多输出）

可用命令

kong check

kong health

kong migrations

kong prepare

此命令用来准备Kong前缀文件夹及其子文件夹和文件。

kong quit

kong reload

kong restart

kong start

kong stop

kong version
