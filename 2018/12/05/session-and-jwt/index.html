жңҖиҝ‘е…¬еҸёеңЁеҒҡзі»з»ҹи®ҫи®Ўзҡ„ж—¶еҖҷжҸҗеҲ°дәҶиҝҷд№ҲдёҖдёӘи®Ёи®әпјҢе°ұжҳҜжңүзҠ¶жҖҒзҡ„зҷ»еҪ•йӘҢиҜҒе’Ңж— зҠ¶жҖҒзҡ„зҷ»еҪ•йӘҢиҜҒпјҢиҮідәҺиҝҷдёӨиҖ…зҡ„еҲ©ејҠжңүе“ӘдәӣпјҢе…·дҪ“еҰӮдҪ•еҒҡйҖүжӢ©пјҢеҸҜд»ҘжҠҠдёӨиҖ…зҡ„зү№зӮ№жқҘеҒҡдёҖдәӣеҜ№жҜ”гҖӮе…¶е®һиҝҷдёӨиҖ…е…ұеҗҢзҡ„зӣ®зҡ„е°ұжҳҜи§ЈеҶіhttpеҚҸи®®зҡ„ж— зҠ¶жҖҒзү№жҖ§пјҢж ҮзӨәе®ўжҲ·з«ҜдјҡиҜқз”ЁжҲ·гҖӮдёҚи®әз”ЁиҝҷдёӨиҖ…е“Әз§Қж–№ејҸпјҢtoken е’Ң session йғҪеҸҜиғҪдјҡиў«зӘғеҸ–д»ҺиҖҢеҸ‘з”ҹCSRFж”»еҮ»пјҢе…Ёз«ҷ HTTPS жҳҜеҝ…дёҚеҸҜе°‘гҖӮgoogleдёҖдёӢеҸ‘зҺ°е…ідәҺиҝҷдёӨиҖ…зҡ„и®Ёи®әйғҪдёҚе°‘пјҢеңЁжӯӨжҖ»з»“дёҖз•ӘпјҢжңүе…¶д»–ж„Ҹи§Ғзҡ„ж¬ўиҝҺз•ҷиЁҖи®Ёи®әгҖӮ

жңүзҠ¶жҖҒпјҲSESSIONпјү

жүҖи°“жңүзҠ¶жҖҒпјҢе°ұжҳҜзҡ„е°ұжҳҜдј з»ҹзҡ„ cookie session пјҢcookieзҡ„иә«д»ҪйӘҢиҜҒжҳҜжңүзҠ¶жҖҒзҡ„гҖӮиҝҷж„Ҹе‘ізқҖйӘҢиҜҒзҡ„и®°еҪ•жҲ–иҖ…дјҡиҜқ(session)еҝ…йЎ»еҗҢж—¶дҝқеӯҳеңЁжңҚеҠЎеҷЁз«Ҝе’Ңе®ўжҲ·з«ҜгҖӮжңҚеҠЎеҷЁз«ҜйңҖиҰҒи·ҹиёӘи®°еҪ•sessionе№¶еӯҳиҮіж•°жҚ®еә“пјҢеҗҢж—¶еүҚз«ҜйңҖиҰҒеңЁcookieдёӯдҝқеӯҳдёҖдёӘsessionIDпјҢдҪңдёәsessionзҡ„е”ҜдёҖж ҮиҜҶз¬ҰпјҢеҸҜзңӢеҒҡжҳҜsessionзҡ„вҖңиә«д»ҪиҜҒвҖқгҖӮеүҚз«ҜйҖҖеҮәзҡ„иҜқе°ұжё…cookieгҖӮеҗҺз«ҜејәеҲ¶еүҚз«ҜйҮҚж–°и®ӨиҜҒзҡ„иҜқе°ұжё…жҲ–иҖ…дҝ®ж”№sessionгҖӮ

дјҳзӮ№

  • зӣёиҫғдәҺж— зҠ¶жҖҒзҡ„йӘҢиҜҒжңәеҲ¶пјҢдј з»ҹзҡ„sessionеҸҜд»ҘзӣҙжҺҘд»ҺеҗҺз«Ҝдё»еҠЁжҺ§еҲ¶дёӢзәҝпјҢеҲ йҷӨsessionпјҢж–№дҫҝе®һзҺ°дә’tзӯүеҠҹиғҪ
  • sessionдҝқеӯҳеңЁжңҚеҠЎз«ҜпјҢзӣёеҜ№жҜ”иҫғе®үе…Ё
  • жңүзҠ¶жҖҒзҡ„sessionеҸҜд»ҘиҫғдёәеҮҶзЎ®з»ҹи®ЎеңЁзәҝдәәж•°

зјәзӮ№

  • жңүзҠ¶жҖҒзҡ„еӯҳеӮЁsessionйңҖиҰҒжңҚеҠЎеҷЁз©әй—ҙ
  • жү©еұ•дёҚж–№дҫҝпјҢйңҖиҰҒsessionеҗҢжӯҘпјҢеҖҹеҠ©redisе®һзҺ°е…ұдә«зӯү

ж— зҠ¶жҖҒпјҲJWTпјү

ж— зҠ¶жҖҒпјҢйҰ–е…Ҳе°ұжҳҜдёҚйңҖиҰҒжңҚеҠЎз«ҜеҺ»еӯҳеӮЁпјҢеҸӘйңҖиҰҒе°Ҷз”ЁжҲ·еҗҚеҜҶз Ғдј е…ҘеҗҺз«ҜжҺҘеҸЈеҗҺпјҢе°Ҷз”ЁжҲ·зҡ„дҝЎжҒҜдҪңдёә JWTзҡ„payloadпјҢе°Ҷе…¶дёҺеӨҙйғЁеҲҶеҲ«иҝӣиЎҢBase64зј–з ҒжӢјжҺҘеҗҺзӯҫеҗҚпјҢеҪўжҲҗдёҖдёӘJWTгҖӮеҪўжҲҗзҡ„JWTе°ұжҳҜдёҖдёӘеҪўеҗҢlll.zzz.xxxзҡ„еӯ—з¬ҰдёІгҖӮеүҚз«Ҝе°ұйңҖиҰҒдҝқеӯҳиҝҷдёӘиҝ”еӣһзҡ„з»“жһңеңЁlocalStorageжҲ–sessionStorageдёҠпјҢйҖҖеҮәзҷ»еҪ•ж—¶еүҚз«ҜеҲ йҷӨеҚіеҸҜгҖӮ

дјҳзӮ№

  • jsonйҖҡз”ЁжҖ§пјҢж–№дҫҝи·ЁиҜӯиЁҖ
  • дёҚйңҖиҰҒеңЁжңҚеҠЎз«ҜдҝқеӯҳдјҡиҜқдҝЎжҒҜ, жҳ“дәҺеә”з”Ёзҡ„жү©еұ•
  • еҚ з”Ёеӯ—иҠӮеҫҲе°ҸпјҢж–№дҫҝдј иҫ“

зјәзӮ№

  • JWTзҡ„еҠ и§ЈеҜҶиҖ—иҙ№CPUи®Ўз®—иө„жәҗ
  • дёҚиғҪж–№дҫҝзҡ„з®ЎзҗҶдјҡиҜқ
  • жіЁй”ҖжІЎжңүеҚіж—¶жҖ§пјҢеҰӮжһңtokenжі„йңІпјҢжіЁй”ҖзҠ¶жҖҒдёӢд»ҚеҸҜзҷ»еҪ•ж“ҚдҪң

зӣёе…іи®Ёи®ә&й“ҫжҺҘ

  • иҒҠдёҖиҒҠJWTдёҺsession
  • cookie-sessionжңәеҲ¶дёҺJWTжңәеҲ¶еҜ№жҜ”
  • еҲ«еҶҚз”Ё JWT еҒҡдјҡиҜқз®ЎзҗҶдәҶ
  • Stop using JWT for sessions
  • иҜ·еҒңжӯўдҪҝз”Ё JWT и®ӨиҜҒ
  • д»ҺжңүзҠ¶жҖҒеә”з”ЁпјҲSessionпјүеҲ°ж— зҠ¶жҖҒеә”з”ЁпјҲJWTпјүпјҢд»ҘеҸҠ SSO е’Ң OAuth2
  • дёҚиҰҒз”ЁJWTжӣҝд»Јsessionз®ЎзҗҶпјҲдёҠпјүпјҡе…ЁйқўдәҶи§ЈToken,JWT,OAuth,SAML,SSO
