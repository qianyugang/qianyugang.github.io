6.UNION

UNION 操作符用于合并两个或多个SELECT 语句的结果集。UNION 内部的 SELECT 语句必须拥有相同数量的列。列也必须拥有相似的数据类型。同时，每条 SELECT 语句中的列的顺序必须相同。UNION命令只会选取不同的值。如果允许重复的值，请使用UNION ALL。

9.Constraints约束

Constraints约束，约束用于限制加入表的数据的类型。可以在创建表时规定约束（通过CREATE TABLE 语句），或者在表创建之后也可以（通过 ALTER TABLE 语句）。主要的约束有NOT NULL、UNIQUE、PRIMARY KEY、FOREIGN KEY、CHECK、DEFAULT
